# ToDo-List

## Dependencies list:

1. nodeJS;
2. express;
3. body-parser;
4. ejs;
5. nodemon;
6. The IDE of your preference;

## Starting the project

After downloading the project and the depencecies, open the CMD, cd the project, type 
nodemon app.js.

On the browser, type localhost:3000.
